Addition of a new language to l10n:
- Potentially useful extension: ARB Editor (Google)
- "When locales specify a script code or country code, a base locale 
(without the script code or country code) should exist as the fallback."
  For example, if lang_en_US.arb exists, lang_en.arb should exist.
  When done, "flutter gen-l10n" should execute without error message.
- If adding a new language to l10n, 
the class l10n_utils.dart can be updated with l10n_utils_update.py (maintenance_utils).
