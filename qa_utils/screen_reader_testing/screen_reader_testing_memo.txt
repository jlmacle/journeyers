Current testing paths:
Windows app: Narrator: Tab/Shift Tab
Windows app: Nvda: Tab/Shift Tab
macOS app: VoiceOver: Caps Lock + q, arrow
Web app: Enter key after "Enable accessibility", Narrator + F6
----------------------

Testing with Narrator:
Ctrl + Windows + Enter: to start/stop Narrator
Ctrl + Arrow: navigation through the page
Narrator + space: to interact with a button, etc.
Ctrl: to stop the reading
Narrator + F1: commands list
Narrator + F2: commands for current item 
Narrator + s (twice and quickly): summary of the page (at least, landmarks, links, headings)
Narrator + F5: landmarks
Narrator + F6: headings
Narrator + F7: all links

Narrator + Ctrl + F12: Toggle developer mode
https://support.microsoft.com/en-us/windows/appendix-b-narrator-keyboard-commands-and-touch-gestures-8bdab3f4-b3e9-4554-7f28-8b15bd37410a


Testing with NVDA:
Ctrl + Alt + n: to start NVDA
NVDA + F7: Links, headings, forms fields, buttons, landmarks 
Tab: navigation through the page
NVDA + q: exit NVDA
https://www.nvaccess.org/files/nvda/documentation/keyCommands.html


Testing with Orca:
orca (terminal): to start Orca
Tab/Shift Tab
Orca Modifier + z: start/stop the structural navigation
Alt + Shift + k: list of links 
Alt + Shift + h: list of headings
Alt + Shift + f: list of form fields
Alt + Shift + b: list of buttons 
Alt + Shift + m: list of landmarks 
Alt + Shift + l: list of lists 
Alt + Shift + i: list of list items 
https://help.gnome.org//gnome-help/a11y-screen-reader.html
https://docs.oracle.com/cd/E53394_01/html/E54806/gliad.html


Testing with VoiceOver:
⌘+F5: to start/stop VoiceOver
Ctrl: to stop the reading
Escape: to close a menu or rotor, to stop an action, or to exit a mode
Assuming the default key combination to have been changed to “Caps Lock“
(VoiceOver utility, Commands):
Caps Lock + k: keyboard help
Caps Lock + ?: user guide 
Caps Lock + Shift + q: to turn arrow-key Quick Nav on or off
Caps Lock + q: to turn single-key Quick Nav on or off
Caps Lock + u + left/right arrow: to navigate items by category
Caps Lock + Shift + Down Arrow: to interact with an area or group 
Caps Lock + Shift + Up Arrow: to exit an area or group
Caps Lock + Shift + Escape: to jump to the app’s top level
Caps Lock + Space bar: to activate an item (button clicked, for example)
Caps Lock + Shift + n: to hear how to use the item 
https://www.apple.com/uk/voiceover/info/guide/_1121.html
https://support.apple.com/guide/voiceover/general-commands-cpvokys01/mac
